# Pickachu - Dino Collector Game

## Description
Pickachu is an agent that creates and manages a small **browser-based 2D game** where a dinosaur moves around a plane to collect objects while avoiding obstacles. The game ends when the player collects 10-15 objects, and the user receives a score based on how fast they completed the task.

## Objective
Create a fun, lightweight browser game with the following features:
- A dinosaur character controllable by the user (keyboard or touch).
- Randomly spawned objects to collect (e.g., fruits, coins, or gems).
- Obstacles to avoid (e.g., rocks, trees, or enemies).
- A scoring system based on time taken to collect all objects.
- A simple UI to display the score and game status.

## How to Use
1. **Run the game**: Open `scripts/dino-game.html` in a modern web browser.
2. **Controls**: Use arrow keys or WASD to move the dinosaur.
3. **Objective**: Collect all objects while avoiding obstacles. The faster you finish, the higher your score!

## Implementation
The game is built using:
- **HTML5 Canvas** for rendering the game.
- **JavaScript** for game logic and controls.
- **CSS** for styling the UI.

## Files
- `scripts/dino-game.html`: The main HTML file containing the game.
- `scripts/style.css`: CSS for styling the game UI.
- `scripts/game.js`: JavaScript file containing the game logic.
